2025年折腾记录

辞旧迎新

每逢新年，总要从告别旧岁开始。

先盘点一下微信联系人，看看这一年新加了哪些朋友，又有谁已湮没在历史的尘埃中，余生恐再无交集。对于新朋友，期盼共创佳绩；对于旧相识，则带着真诚的祝福告别，愿他们前途无限，平安幸福。

接着清理iCloud备忘录，整理去年的日记，并梳理散落在硬盘各个角落的网络素材——那些收集来的趣图、段子，以及无数的音乐和视频文件。这项工作虽费时费力，但对未来的资料检索至关重要。

当然，还有一件头等大事：把To Do List的年份从2024改为2025。毕竟，未竟的事业要在新的一年里继续传承。

既是新年，聚会自然必不可少。中国人的热闹总是以饭局开场，觥筹交错间，祝福与期待萦绕在升腾的氤氲热气中。这种温馨场面，比老外们在Party上凑成一堆干站着喝酒聊天要有意思多了。

步入新的一年，感觉身体状况又在下坡路上滑了一截。虽然努力通过改善膳食和锻炼来保养，也尽量避免熬夜，但随着年岁增长，各种小毛病依然层出不穷。于是趁着公司政策的春风，把大病保险足额买齐了——毕竟咱们北美不比东瀛，万一家里的男人不在了，房贷还是得靠未亡人继续供的。

政经风云

开年没几天，总理贾斯汀·特鲁多——也就是华人口中的“小土豆”，在党内压力下宣布辞去党首职务。虽然这是众望所归，但他的突然离去给加拿大政局带来了新的变数。

与此同时，南边那位特朗普先生风格依旧琢磨不定，他对加拿大突然提出的领土主张与关税大棒，竟成了“神助攻”。在双重压力下，本已快走进历史的自由党实现了惊人逆转，不但在联邦大选中击败了数月前民调遥遥领先的保守党，还差点拿到了多数党席位。可见世事难料，不到最后一刻实难见分晓。

股市方面的剧情则更为精彩：先是4月3日，美国总统宣布向各国加征对等关税，股市暴跌；次日，中国宣布反制，股市继续下挫。市场在恐慌几天后消化了情绪，直到9号美股狂泻后触底反弹。最终在5月12日，随着中美就贸易战达成协议，股市迎来暴涨。

在这场暴涨暴跌的过山车中，究竟有多少人靠内幕消息攫取了惊天财富，恐怕永远是个历史谜团。不得不感叹美国真是“自由”，像特朗普这种明晃晃操纵股市的行径，若放在韩国，恐怕早就吃上牢饭了。

作为普通百姓，我们除了在大潮中随波逐流，也就只能顿足捶胸，遗憾那泼天的富贵为何没降临在自己头上。

第九艺术

Steam的新年折扣向来是重头戏。趁低价入手了《Anno 1800 - Definitive Annoversary》和《Far Cry New Dawn - Deluxe Edition》。前者是不折不扣的“时间黑洞”，虽音画俱佳，但受限于游戏机制，即便是祭出了单机神器Cheat Engine，依然要搭进去无穷的时间精力，让我这样的上班族倍感压力。后者则是育碧典型的流水线罐头，在收集到静音远程武器的那一刻，基本就可以纵横天下无敌手了。

玩《Anno 1800》时，为了追求视觉享受，我曾把主机搬到电视旁，试图用55寸4K大屏优化体验。但理想与现实总有差距：画面虽大，但这就要求我不停地扭头扫视屏幕，对颈椎负担极大，且电视的高亮度也比显示器更刺眼。坚持了约摸一周，主机还是搬回了原位，游戏终究还是在24寸显示器上玩得舒服。

不过，有些游戏天生适合电视。比如去朋友家做客时体验了Switch上的《Overcooked（胡闹厨房）》，效果很棒。我手上恰好有台Xbox，本想趁微软打折买点游戏，奈何微软商城与我的信用卡似乎“八字不合”，接连换了几张卡、几个平台和浏览器，均提示付款失败。

无奈之下，只好回归熟悉的Steam平台，趁新年大促一口气买下了《Overcooked》和《It Takes Two（双人成行）》。接着又在Amazon低价入手了Xbox Core Wireless Gaming Controller。两副手柄通过蓝牙连接笔记本，再外接电视，总算实现了“用Xbox手柄在大屏幕玩游戏”的愿景。

《Overcooked》难度可调，很快便通关了；但《It Takes Two》后期难度陡增，卡关一阵子后便将其束之高阁。

为了不让新买的手柄吃灰，我又在Amazon买了一款热销的TP-Link蓝牙适配器插在台式机上（台式机无自带蓝牙），试图用手柄玩《如龙》。但体验有些微妙：玩《如龙7》、《如龙8》这种回合制游戏尚可，大不了慢慢选招；但《如龙7外传》和《如龙0》是即时战斗，新招式层出不穷。对于我这种隔几天才玩一会儿的健忘中老年玩家极不友好，经常玩到后期就把招式忘得一干二净。

况且手柄按键繁多，激战中一时想不起键位，因按错而败北的挫折感简直无以复加，令人气馁。所以没过多久，我还是切回了键鼠模式——战斗再复杂，配合“拜年剑法”和Cheat Engine，总能看到过关动画。

这一年下来，陆陆续续通关了《Far Cry New Dawn》、《看门狗2》、《如龙8》和《魔法使之夜》。我的老机器虽配置于2018年末，但依然发挥稳健，令人甚感欣慰。

银行二三事

生活在北美，Costco是绕不过去的必需品。会员费年年涨，也只能硬着头皮续。地税更是如此，自八年前搬来现址，地税已累计上涨33％。这种每年都要被扒层皮的事，也只能乖乖奉上血汗钱。按理说人口增加、市政收入变多，摊到每个人头上的税应该减少才对，或许是人越多市长担子越重，项目支出水涨船高，地税也就坐上火箭一飞冲天了。

拜“花呗”所赐，国内朋友对自动扣款早已习以为常。但我对此曾长期抵触，总担心机构会在我不注意时掏空家底，因此多年坚持“见账单，手动付”。直到有一次统计电费，发现因忘记付款被罚了滞纳金，自责之余赶紧亡羊补牢，开通了自动预付费。

同样，申请了C银行与Costco联名的MasterCard后，我也忘了在App里开启自动还款，导致首月账单逾期。好在发现及时，联系客服说明情况后，银行大度免除了滞纳金。

感觉商家有时就在赌消费者的记性。比如我不小心误开通了一个中国移动的服务，刚开通就点了取消，但App提示不满30天不能取消。30天后我当然忘了这茬，结果又被阴险的商家叼走一块肉，心痛不已。

今年受公司委派去德国出差，因准备不足，到酒店才发现没有适配北美电器的插座。恰逢周日商店歇业，最后不得不在火车站小店以40欧元的“天价”买了个做工粗糙的转换插头——也就是淘宝上几块钱的货色，算是一次难忘的教训。

到了欧洲，刷卡也成了问题。尽职的银行利用大数据帮我们抵御盗刷风险，但这就像双刃剑：防住了小偷，也防住了户主。在支付高昂漫游费打了一通漫长越洋电话后，信用卡终于解锁，但很快发现App弹窗金额与账单不符。咨询客服才知，那是事后额外加收的2.5％外币手续费。

买小件不觉得，但替同事代付了一只LV手包后，这2.5％就显得颇为肉疼了。于是我开始搜寻免外币转换费的信用卡。市面上确实有：Rogers联名卡虽好但我非其用户；剩下的选项便是EQ Bank、WealthSimple和ScotiaBank。

ScotiaBank的Scene积分当年很诱人，但如今贬值严重，吸引力大减。WealthSimple将对资金不足的客户收月费，我本冲着免费来的，怎肯花钱？于是果断选择了EQ Bank。老实讲，EQ除了App略显粗糙、通知全靠邮件有点烦人外，没什么大缺点。而且它是先充值后消费，不会透支，平时少放点钱就不怕盗刷，反成了一项优点。

常坐飞机的朋友向我安利Amex信用卡的福利，从提前值机到贵宾室确实诱人，但年费也着实昂贵。我自认飞行频率不高，很难赚回年费，还是留给那些精力旺盛、满世界飞的朋友享受吧。

国内许多人抵触信用卡，多是担心安全性及盗刷后的责任认定。好在这里的银行处理盗刷还算可圈可点。鄙人的美元信用卡平时仅用于ChatGPT月费和购买机票，某天却忽然被盗刷了几笔Uber Eats。可气的是，这群蟊贼竟在多伦多使用我的美元卡点外卖，这显然不是理智之人所为。

联系C银行处理Dispute（争议）流程倒是不难，致电Fraud部门即可。但外包的南亚业务员出了点岔子：一共四笔盗刷，她却搞错了其中两笔的金额，导致我不得不再次致电更正。

除却这些小插曲，经历约一个半月的漫长等待，审核终于结束，四笔盗刷悉数免除。银行下发了新卡，激活后一切如常。

房贷

与南边那个西方大国不同，加拿大的房贷利率每隔几年就要重谈，即便不换银行，合同期也不能超过五年。虽有听起来无限期的“无固定利率合约”，但终究不像美国人那几十年不变的利率让人安心。

房子虽买没几年，房贷却已续了好几轮。上次续约恰逢我去日本度假，本以为耽搁几天无妨，谁知短短几周利率便从6点几飙升至7点几，只好捏着鼻子忍受了一段高息时光。

好在按规定，到期前6个月即可提前续约。一到最早可续约日，我便赶紧联系银行重谈。趁手头现金充沛，我和R银行商量提前还款，但银行深谙“放长线钓大鱼”之道，只允许我在续约时最多偿还10％的本金。考虑到银行利息之高让任何学过小学数学的人都如芒在背，哪怕只还10％也是相当可观的，我便同意了。

接下来的事却让人大跌眼镜。我的资金在C银行，为筹措R银行的扣款，特意从T银行办理了汇票。未曾想这笔钱竟非实时到账，在C银行账上滞留了整整一周。在那个关键时刻，我C银行借记卡余额为0，导致R银行扣款失败。眼看信用即将受损，我急忙联系C银行客服，并在建议下找亲友紧急筹款以备不测。

好在人品尚佳，几位朋友仗义疏财，帮我避免了二次划款失败。后来在C银行客服经理协助下，先解锁了一部分资金解了燃眉之急。有惊无险度过危机后，我发现朋友的e-transfer限额比我高一大截。找客服经理理论，大家同在一片青天之下，何以待遇悬殊？客服先是狡辩一番，核实后还真帮我也争取了同样权益。可见即便在太平洋这一侧，也是“会哭的孩子有奶吃”。

新的三年固定利率降到了3.96％。虽比不上朋友圈里高人谈下的利率，但“山外有山”，平衡时间精力与效益，这个数字我基本满意。

人类从未真正战胜过蚂蚁

年末，ChatGPT根据我一年的对话生成了一张图：我坐在电脑前，屏幕上是Python图标，桌面上爬着一只小蚂蚁。

说来讽刺，疫情刚爆发时，我就在家捣鼓Python、与蚂蚁抗争。五年过去了，年度折腾主题居然还是Python和蚂蚁。

是的，蚂蚁死灰复燃了！

作为近600户居民的公寓住户，楼内管道纵横、缝隙错综复杂。法老蚁这种外来入侵物种在此如鱼得水，繁衍迅速，已成北美著名公害。

受限于环保法规，即便是政府持牌的杀虫人员，手里的药剂毒性相比国内配方也是微不足道。小蚂蚁们很快就对市面上的杀虫剂与毒饵产生了抗药性。起初布置的药饵还能管个把月，但很快它们便卷土重来，愈演愈烈。

市政人员来了一次又一次，门缝墙角的药剂从透明变黄。毒饵陷阱遍布家中，但蚂蚁视若无物，依然频繁出没于浴室和厨房。纵使它们不咬人，但看着活物在家具上爬行，总让人毛骨悚然。

好在我们中国人背靠母国和淘宝，依然可以搞到毒性不打折的“呋虫胺”。按说明兑水喷洒一遍，只等了一晚，肆虐足月的蚂蚁便消失无踪。但愿这次能保半年清静吧。

AI

@信号与噪声：普通人，但凡重大决策前，把自己的想法多跟几个 AI 聊聊，都能避免 90% 以上的坑

AI爆发已有几年。早在2023年我就尝试用Stable Diffusion文生图，还一度想为了提升出图速度升级显卡。但AI能做的远不止于此，随着头部厂商疯狂砸钱，AI的智能水平肉眼可见地提升。

几年前程序员还能像马车夫嘲笑蒸汽机车那样嘲笑AI的代码，现在显然笑不出来了。前几年咨询AI时它还常犯错，今年准确性已大幅提高。Stack Overflow访问量的瀑布式下滑也佐证了这一点——当然，除了AI冲击，老用户倚老卖老导致社区氛围对新人不友好，也是阻碍其发展的重要因素。

年初受中国AI厂商DeepSeek冲击，英伟达和微软股价暴跌的景象历历在目，Meta开出上亿薪酬抢人的新闻更是刷足关注度。虽然不少国人甚至专家视AI为泡沫，认为是虚无的经济支撑，但我作为高频用户，常被AI抛出的金句惊艳——那些冰冷算法组合出的概率文字，竟能如此深刻。

AI的实战能力也让人刮目相看：内子的Oral-B电动牙刷罢工，AI指导我按压某两个按钮成功将其复活；至于帮我改进代码、配置服务器、生成FFmpeg命令、制作手机壁纸等工作，更是数不胜数。

缝缝补补又三年

公寓开发商原配的天花板灯具十分昏暗，内子对此不满已久。今年总算换成了佛山产的LED灯。新灯不仅亮度大增，而且将原来的暖黄光换成了日光色，整个客厅顿时耳目一新。

趁热打铁，我又把家里几个常年漏水的水龙头阀芯换了。唯一糟心的是，换完浴室阀芯才从同事处得知，阀芯厂家Moen提供终身保修，去Home Depot打个招呼就能免费领。我试着把收据发给Moen，客服表示无法报销，但出于同情可以寄一个新的，只需我付运费。哎，怪自己功课没做足，只能当花钱买教训了。

此外，还用补缝剂填补了洗手池、浴缸等处的墙缝。房子住久了难免开裂，担心渗水霉变，请人太贵，索性自己动手。

某天无意发现带去公司用的X201有个声道没声音，仔细检查发现是扬声器坏了。淘宝下单笔记本喇叭组件，拆机更换，轻松搞定。

或许家用电器也步入了浴盆曲线（Bathtub Curve）的故障高发期。某晚微波炉罢工，在AI协助下判断是保险丝问题。AI甚至通过照片帮我定位了保险丝位置并确认了件号。最终只花几毛钱换了个保险丝，省下了买新微波炉的几百刀。

洗衣机报故障码，结合现象判定是冷水进水阀故障。看了YouTube维修视频，从Amazon订购新阀，找朋友帮忙把摞在上面的烘干机抬下来，更换进水阀后故障排除。

汽车电瓶也在某天下班时失效，幸好同事有Booster帮我脱困。随即去Costco买来新电瓶，拧几个螺丝便更换成功。记得车商曾发过召回通知，称电瓶安装不当有起火风险——我换的当然不会有这问题，毕竟咱也是带着专业常识操作的。

手头这台M900 Tiny购于2015年，距今整整十年。它的i7-6700T CPU即便在今天也能流畅播放B站视频，Windows 7系统依旧丝滑。我离不开ACDSee 3.1和拼音加加，平时用的VS Code 1.70.3和Python 3.8.10运行流畅，FFmpeg Essential版也表现良好，实在没有升级理由。虽然逢年过节看到新电脑广告会流口水，但一想到买来新设备老设备只能吃灰，便强忍住了冲动。

折腾无止境

为了凑Amazon满35免运费，顺手买了一台打折的TP-Link TL-SG108交换机。原先设备接路由器也能用，但随着电视、游戏机和多台PC接入，路由器自带的四个口显得捉襟见肘。这八口交换机一买回来，插上现有设备立刻占了七个口，看来还是买保守了，该买24口的。

家用到底需不需要交换机？见仁见智。我觉得更多是心理爽感。毕竟内网那点数据量，路由器也能扛。至于无线带宽有限，只要有耐心，数据总能传完。但在心理上，觉得有线设备走交换机，路由器专心服务无线设备，速度或许会有提升吧。

刷Amazon时看到罗技K400 Plus无线键盘打折，它集成了一个小触摸板，可作为ThinkPad无线键盘的“平替”。手感虽一般，但接在电视上搜视频比用遥控器选字快多了。不过电视除了屏幕大，体验被PC完爆：PC浏览器可装Ublock Origin去广告，有yt-dlp下载视频，搜索跳转也方便。正因我看电视极少，这键盘并未带来预期中的提升。

多年前买过GoPro 4，很快就吃灰了，我实在不是那种愿意脑袋顶着摄像机出门的人。但看同事玩大疆运动相机风生水起，恰逢发了奖金，便入手了DJI Osmo Action 5 Pro Adventure Combo。试玩后发现和GoPro 4大同小异，除了内置存储免去买卡、软件好用点、传输快点外，本质没变。这东西还得看人，像我这种不爱出门也不爱运动的，简直是明珠蒙尘。

为了追求“咖啡梦”，入手了DeLonghi EC9255.M咖啡机和咖啡豆。但我终究怕麻烦，咖啡又不能多喝（心脏受不了），为了喝一杯Espresso要费力清洗一番，实在难以坚持。于是它也被遗忘在厨房角落，仅在客来时才启用。

买了个罗技G304鼠标，手感一般，舒适性不如微软大馒头（Microsoft Sculpt Ergonomic），可惜后者停产了。备受好评的Master 3S我也有，但这鼠标配套软件臃肿不堪，前阵子还因证书过期导致Mac用户功能失效，实在无语。好在G304即插即用，目前暂且用着。

AWS EC2一年试用期满，素知AWS乱收费恶名，我谨慎清点了服务，停掉EC2，换用更便宜的Lightsail。反正我只在上面搭了个Nginx放静态网页，以及Tailscale的DERP中继节点，不吃性能，原先的EC2有些浪费。

起初担心1TB流量不够，多花2美元选了2TB的配置，结果一个月连100GB都跑不到。可见本该买最便宜的配置，但AWS不支持一键降级迁移。考虑到域名、Cloudflared DNS代理、Nginx和Tailscale这一套配置太繁琐，懒得再折腾，也就睁一只眼闭一只眼了。反正这点小钱相比日常支出的大头，连浪花都算不上。

Cloudflared Tunnel我也体验了一下。家里B公司的宽带不知何故竟给了公网IP，但从国内访问速度感人，加上我对暴露家庭IP心存恐惧，体验几分钟便草草收场。

今年Debian 13终于出了稳定版，我把内网服务器和VPS都升级到了13。遇事不决问AI，报错信息扔给它很快就有解，如今折腾Linux比过去容易太多。

Tailscale开发组很勤奋，版本更新频繁。但我给服务器系统打了补丁后DERP挂了，才知服务器Tailscale版本须与DERP一致。好在AI在手，无论是Docker配置还是Go源码编译，AI都能带我如履平地。Grok、Claude、ChatGPT和Gemini几大巨头轮流用，免费额度足矣。

这一年的主题既是蚂蚁也是Python。消灭完蚂蚁，继续钻研Python，处理txt爬虫程序中的特殊字符和换行。试用了代码格式化工具Black和Ruff，各有千秋。至于许多人推荐的VS Code主题“Monokai”，我实在欣赏不来，或许眼睛已习惯了微软精心调研后的默认配色。

以AI为师，我还学习了网页和JavaScript知识，改进图片加载。学爬虫得先懂网页结构，也是在这一年，我踩了Cookie的大坑，才了解Cookie获取与User Agent的强关联。哎，野路子出身的外行，比起科班生确实吃了不少闷亏。以后有空想系统过一遍CS大学经典课程，为业余爱好添砖加瓦。

白月光的 ThinkPad

曾经有一份真诚的爱情摆在我的面前，但是我没有珍惜……如果非要把这份爱加上一个期限，我希望是一万年！

央求多年，姐姐终于把她大学时用的 ThinkPad X61 送给了我。这台当年售价近万的“梦中情机”，如今终于摆在了我的案头。当年囊中羞涩，我自然是没有亲近它的机会，人生中第一台真正属于自己的 PC 是大学时购入的 ThinkPad R61e 7649AG2——联想试水市场的入门级产品，媒体甚至起了“比神舟还便宜”的标题。眼前这台 X61 与我那台 R61e 的出厂日期前后仅相差数月，看着这同样制造于 2007 年的机器，那 4:3 的绝版屏幕、几乎顶着边框的键盘，以及那个年代独有的大键程手感，无不令人心潮澎湃。

这台 X61 搭载的是 T8100 CPU，而我那台 R61e 只是赛扬 540；X61 轻至 1.65KG，R61e 则重达 2.93KG。或许正是因为第一台笔记本虽然厚重但皮实耐用给我留下的良好印象，才让我几年前毫不犹豫地入手了同样重达 2 公斤多的 15 寸 T15p。事实上，大学时期的我体力充沛，背着 3KG 的笔记本上课并不觉得累——毕竟高中的书包可比这沉多了。反倒是 15 寸大屏带来的视觉享受实实在在，我也正是靠着那台性能堪忧的笔记本完成了学业，并保持着年级第一的辉煌直至出国。当然，这都是后话了。

然而，随着对这台小笔记本的深入折腾，我愈发觉得“距离产生美”，有些东西或许还是留在记忆中最好。这台机器的缺点着实不少：比如扬声器是单声道的，听个响都费劲；右掌托下方就是硬盘，运行时发热巨大，手搭在上面简直像垫着个暖宝宝。我接手时它装的是 Win10，对于 Core 2 时代的 CPU 来说，如此“超前”的系统实在太勉为其难。我本想换成 Linux 曲线救国，却不知是安装过程中触发了显卡固件 Bug，还是久未启动导致排线故障，屏幕下方出现了一条两三个像素宽的常亮线，进 BIOS 也无法消除。考虑到这种古董机已无修复价值，只能听之任之。更令人沮丧的是，费尽周折装上的 Linux 体验也不尽如人意，无论是 Debian 还是 Ubuntu 都卡顿明显，看来现代 Linux 的桌面环境对这颗古老的 CPU 来说，还是太“重”了。

既然 Linux 走不通，还得回归 Windows。为了让它重现当年风采，我决定像对待 R61e 那样安装 XP 系统。先是找来著名的《上海政府版 XP》，安装过程虽顺利，但后续配置却是噩梦。该机型早已停止官方支持，联想自带的 System Update 也已断连。海峡对岸的 Tpuser 站长虽写过手动安装 XP 的指南，由于时过境迁，文中的链接早已失效。尽管我搜遍全网，手动找来各种配套软件，但大半要么无法安装，要么装完报错。比如电源管理 Power Manager 和网络管理 Access Connection，驱动不匹配开机就崩；再比如当年机械硬盘必装的 Active Protection（硬盘保护），费了九牛二虎之力装上去，软件竟报错说检测不到传感器，建议我卸载。

被驱动折腾得心力交瘁后，我决定寻找系统镜像恢复出厂状态。经一番搜索，终于在 archive.org 上找到了好心人上传的 X61 原版镜像。本以为用 Rufus 把镜像写入 U 盘就能引导，谁知联想当年不知用了什么特殊引导机制，换了几个镜像都卡在同一处。最后还是抱着死马当活马医的心态，找来足足 7 张 CD 空盘，配合外置 USB 光驱，才终于把 X61 的 XP 系统恢复成功。虽然预装软件版本老旧，但至少都能正常运行了。

其实 X61 和 R61e 理论上共享同一套恢复逻辑，但时隔近二十年，我当年刻录的 R61e 恢复光盘早已不知所踪。记得我那 R61e 虽预装 Vista，但 512MB 的内存跑 Vista 卡得令人发指，到手没两天就被我换成了 XP。说来也怪，当年 XP 陪伴我度过了整个大学岁月，玩游戏、发短信、写日记、做作业、复习考试，完全得心应手。但今日重温，却只感痛苦不堪。虽然费力安装了 Legacy Update 打满补丁，又搜罗了各种 .Net Framework 运行库，但时代的鸿沟依然难以跨越。首当其冲的便是浏览器体验。

能找到的支持 XP 的 Firefox 和 Chrome 早已停止对现代网页的支持，甚至连 Github 的下载链接都打不开。好在偌大的互联网上总有人试图抢救旧时光，开发了一个名为 Mypal 的浏览器项目，勉强支撑起基础的网页浏览。至于字体，XP 时代的点阵宋体与 Win7 以来的微软雅黑相比，只能用“折磨”来形容。更让我不习惯的是对开始菜单搜索功能的依赖——在 Win7 以后，点开菜单输入首字母直接跳转程序已成肌肉记忆。以至于在 XP 上，我习惯性地输入 "p" 想开画图（Paint），弹出的却是电源管理（Power Manager）的启动界面。

此外，联想在那个年代的键盘布局上别出心裁，把后来放置“翻页键”的按键设为了“后退/前进”，使用中屡屡误触，打断思路，烦不胜烦。而 XP 上又没有 Win7 那般好用的 SharpKeys 改键工具，只能进注册表手动修改键值。至于磁盘管理，XP 也不具备 Win7 以后那种无需第三方工具即可调整分区大小的便利功能。

硬件方面，这台机器的电池已近报废，寿命不足全盛时期的 40%。考虑到它是 18 岁高龄的古董，且根据当年的评测文章，其新机续航也不过短短 2.5 小时，甚至不如我现在的 T15p。因此无论从哪个角度看，花钱换电池或升级都毫无性价比，就让它作为情怀静静展示吧。

既然是我众多 PC 收藏的一员，自然要想方设法让它加入局域网。但考虑到 XP 的安全性，生怕它中招后拖累全家，我不敢让它直连 Dell 服务器，更不敢为了兼容它而让 Win10 开启不安全的 SMB 1.0。最终的解决方案是：在 Dell 服务器的虚拟机 Linux 中搭建 Samba 共享服务，以此为中转，实现了 Windows 10、11、7 与这台 XP 古董机的文件互通。

重游雍和宫

拜年假所赐，又回了趟北京。站在阔别十多年的雍和宫，看着顶礼膜拜的信徒和烟雾缭绕的殿宇，顿时百感交集，赋文一篇，作为今年回顾的结语——

余久寓海外，目睹教堂林立，每见辄念及新约所谓“神爱世人”。然东西万民，皆食五谷，杂处红尘，孰无惑障，欲叩天阍以求解脱乎？古人云：“天皇皇，地皇皇，吾心惶惶，求神问卜。”斯乃人之常情也。
昔余壮年，曩游雍和宫，时方青春，意气方遒，仕途通达，囊中金玉，以为天地之大，无不可至。今岁三十有八，齿危发白，形容憔悴，恰似韩子所谓“发秃齿豁，行步龙钟”。身病心劳，夜不成寐，家庭琐事，纷扰不宁。吾妻病屋宇之隘，欲迁广厦，然金珠昂贵，力有不逮。夫妻每因纤芥之事，争端蜂起，徒增烦忧。父母春秋已高，余远在异邦，未能晨昏定省，尽人子之孝。内不能谐琴瑟之好，外无以承欢膝下，中心怅然，郁结难抒。
今重游斯宫，殿宇巍峨，层峦叠翠，香烟袅袅，梵音缭绕。自北朝以来，香火不绝，士女如云，祈福禳灾，愿心千万，然果有几人得偿所愿乎？庄子云：“人之生也，与忧俱生。”斯言不虚。余观熙攘人群，焚香顶礼，各怀心事，或求富贵，或祈安康，然世人之惑，果能尽解耶？
嗟乎！人生在世，苦多乐少，恰似庾信之叹：“人生如寄，忧患实多。”佛氏虽云慈悲，然楞严有言：“自心自度，非佛能渡。”余虽凡夫，焉敢望脱苦海？惟愿身无疾患，家室和睦，父母康宁，斯足矣。
兴尽悲来，徘徊殿前。愿天皇地皇，鉴我微诚，赐以清宁，消我烦忧。
